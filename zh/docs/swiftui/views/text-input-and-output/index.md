# 文本输入和输出

> 显示格式化文本并从用户获取文本输入。

## 概述

要显示只读文本或与图像配对的只读文本，请分别使用内置的 Text 或 Label 视图。当您需要从用户收集文本输入时，请使用适当的文本输入视图，如 TextField 或 TextEditor。

![text-input-and-output-hero](https://docs-assets.developer.apple.com/published/a8d0f352ff6dabd1c3d1e3fdfb577250/text-input-and-output-hero%402x.png)

您可以添加视图修饰符来控制文本的字体、可选择性、对齐方式、布局方向等。这些修饰符也会影响其他显示文本的视图，例如控件上的标签，即使您没有定义显式的 Text 视图。

有关设计指导，请参阅人机界面指南中的排版 。
